Password Check Web-App

The PasswordCheck is a full-stack web-app that allows users to analyze the strength of a given password anonymously. It also allows storing the results into a Mongo database (stripped from all sensitive information) to conduct analyses.

The app uses a NEMP Stack: NodeJS, Express, MongoDB, Polymer.
